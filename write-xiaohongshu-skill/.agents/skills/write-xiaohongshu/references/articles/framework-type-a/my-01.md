# 产品经理必备的3个思维

**分类**：framework-type-a
**主题**：产品思维
**风格**：干货类
**来源**：小红书 @夏树
**添加时间**：2026-01-30

---

做产品这么久，发现很多新人都在埋头干活，却忘了抬头看路。今天分享3个最重要的产品思维，帮你少走弯路💡

**1. 用户思维**

不是说"我觉得用户需要什么"，而是"用户真正需要什么"。

举个例子：我做过一个功能，自认为很完美，结果上线后数据惨淡。后来才发现，用户根本不需要这么复杂的东西。

记住：你代表不了用户。

**2. 数据思维**

别凭感觉做决策，让数据说话。

每次迭代前问自己：这个改动能带来什么价值？怎么衡量？

不是所有改动都要做AB测试，但至少要有预期目标。

**3. 成本思维**

资源永远是有限的。

做什么、不做什么，比怎么做更重要。学会做取舍，把资源花在刀刃上。

---

产品思维不是一天养成的，但可以从现在开始刻意练习。
