# 大厂绩效考核真是一场提心吊胆的狼人杀😨

**分类**：game-metaphor
**主题**：绩效考核、狼人杀比喻
**风格**：游戏比喻分析类
**来源**：小红书 @夏树
**添加时间**：2026-01-30

---

这几周各家大厂年中绩效就要开奖了。和前同事聊，都感慨幸亏我们已经从大厂出来了，不然这会儿正为悬而未决的绩效焦虑。这半年一次的绩效，真像一轮轮狼人杀

🔖游戏开始，领导请发牌
牌型并非随机，一开始就定好了。同一个团队，身份天差地别，好的是嫡中嫡、core，是狼人、神牌，手握权力与信息，有好项目、资源多，易出成绩；差的是牛马、透明人，是村民，没资源没机会，汇报也轮不上，都是嫡系或者虚线去汇报；交际花、关系户，像白痴，有免死金牌，没人敢惹

🙋玩家请发言
发言环节2个季度，所有人踊跃发言，赢取领导认可好被发金水。在各种汇报工作场合，周会、周报、写绩效自评，拼力表现，暗自较劲。你讲新发现，我便讲拿了核心指标收益；你做了个规划，我也做新规划；你忙于做好本职工作，最内卷的嫡中嫡，为了扩大scope给自己招人，不光卷别的团队，还来卷你的活

🙈天黑请闭眼
下一轮被刀的是谁？绩效期的前几个月，众人就根据苗头猜自己在领导心中的顺位。如果领导经常问你项目、向上汇报，恭喜，你的顺位靠前；如果领导老批评你，或冷处理，那你危险了。一个常见现象：绩效前几个月，同事们接二连三离职。原本觉得安全的同事，看到后面的人走了，也赶忙跑路

🙅‍♀️天亮了，所有玩家睁眼
绩效不会公开，但大家私下一交流，就知道昨晚走了几个。团队业绩蒸蒸日上时，没有差绩效，昨晚平安夜。团队业绩差时，末位淘汰名额变多。这一轮暂时安全的人，可以喘息几口，等下一轮天黑请闭眼…

🤫
大厂的绩效考核与狼人杀最为相似之处，在于时刻提着心，适应这套残酷的游戏规则——高人才密度、高考核标准、高淘汰率，加之看似扁平实则复杂的层级，看似透明实则处处不透的信息差，不幸者迅速出局，侥幸存活者也被困于这看不见的围城里。

我是夏树，前大厂产品专家，真诚分享
校招、绩效、面试、landing…持续更新
