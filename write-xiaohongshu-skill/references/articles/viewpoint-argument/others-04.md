# 实习生进步最快的方式就是逼自己写小红书

**分类**：viewpoint-argument
**主题**：输出倒逼、表达能力
**风格**：观点-方法类
**来源**：别人
**添加时间**：2026-01-30

---

写小红书的目的不是为了涨粉和做自媒体，是通过公开的输出倒逼自己去保持思考和输入

因为即使再多人反复告诉你，在实习当中天天写一点东西去总结自己的思考很有用，但你是坚持不下来的，因为没有奖励、反馈和监督

很多人说自己表达能力不够好，面试能力不知道怎么提升，如果你不是天生脑子反应就比较快，很多话能张口就来，那么训练表达能力最好的办法就是保持输出

当你开始在小红书上面发自己在实习当中总结的东西的时候，就会发现 如果你写的好、写的对，会有人支持你、催更你，这是奖励和监督

如果你写的不对，会有人跟你杠，会有内行人给你提供别的视角 启发你去思考，这是督促和保持输入

这两种都是实习生在写多少私密的个人飞书文档里没有办法获得的

实习当中，很多思考和灵感都是转瞬即逝的，如果当时不及时总结和梳理是会忘记的。明明这个点你也思考过，但最后面试官问到你的时候，就是说不出口

原因在于当时你想到的时候没有通过书面的形式把它系统梳理一下，所以不管别人问你什么的时候，都是支支吾吾 没有办法顺畅地说下来

写的东西可以包括这几类：

1️⃣ 从公司内部里面学到的知识性的东西。
当你从公司内部文档里面学到一些专业的概念和知识的情况下（比如分层理论、AB test、六大人群），可以通过"是什么-为什么-怎么用-举个例子"的结构把这个知识梳理清楚

没有推崇做知识的搬运工的意思，想说的是这种知识是需要加入一些你的点评或思考的，不知道写什么可以参考cc老师的 @CC里（可内推版） 广告知识系列

2️⃣ 各种方法
包括自己是怎么拿到每份实习的、从mentor或者是身边的同龄人那里学到的好用的办公技巧、怎么从一个行业跨到另一个行业的

实习当中的某个case里面你是怎么做到这么好的、mentor是怎么拿到成果的，有没有什么优化空间，如果再来一次你会怎么做

今天mentor夸了你，是哪里做的好且这个好有没有可以让别人借鉴的东西；今天mentor指出你的一些问题，这个问题是真实的还是被bully了，到底需不需要改进，如果要改的话从哪个方面改

这个部分是把你如何做到xx的过程梳理清楚

有些话就是写的多了才能张口就来的。到面试的时候，你回答的就不是临时想到的答案，而是你已经写过 梳理过 甚至被很多人验证过的成熟观点
