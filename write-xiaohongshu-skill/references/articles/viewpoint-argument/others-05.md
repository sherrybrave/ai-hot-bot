# 只要不熬夜+密集做事就会超级满电飞速进步

**分类**：viewpoint-argument
**主题**：精力管理、个人发现
**风格**：个人方法论类
**来源**：别人
**添加时间**：2026-01-30

---

实习的时候不懂为什么每天都那么高精力，能把排山倒海的任务都安排地井井有条，一件一件地推进下去且全部拿到好的成果，后来才发现是因为只要不熬夜+密集做事就会能量满电

我从最后一段实习开始晚上基本不熬夜 白天很早起来，因为我发现只有早上5点到10点 这个时间是融入社会时钟之前的个人时间，我不愿意失去这个个人时间，也很珍惜这段完全属于自己 不被外界打扰的时间

当我坚持不熬夜和和早起之后发现，即使总睡眠时长很短，但也完全起到修复身体和大脑的作用了，前一天的脑雾也缓解了很多，每天早上醒来都能获得一大段纯净的注意力

并且由于不熬夜不失眠，所以也不会让没睡好导致的疲惫影响做事的效率，白天不管做多少事情都情绪很稳定，意志力损耗很少

再加上密集做事，一件一件地往下推，很容易进入深度工作甚至是心流状态，以及如果产出一些可见的进展的时候，这种成就感会继续反哺动力

当我的脑子被手头的事情占满的时候反而没时间内耗，精神状态越来越好

每天这样过下来，晚上都会有一种恰当疲倦后的满足感，我知道我做的事情都在慢慢推进，明天可以继续耕耘等待成长

这个循环就这么持续运转下去了

很多老师私信我说 白天实习即使做的是一些简单的dirty work，一天下来也会很累，晚上根本没有办法集中精力去学习新的知识、写总结文档，导致最后产出很少

说实话，我也是这样的。不管白天做什么到了晚上都会疲惫，每天精力最好脑子最清醒的永远是早上刚醒来的时候，所以我把学习新的知识、写总结反思文档和规划一天的TODO这些事情都放在早上，就不用消耗意志力硬抗了

如果你现在实习很疲惫感到很消耗心力，不妨试试不熬夜+早起，刚开始会很痛苦，但最初几天早起的不适感很快会被一整天高度的掌控感取代

永远把最好的精力留给自己，这件事不会错的
