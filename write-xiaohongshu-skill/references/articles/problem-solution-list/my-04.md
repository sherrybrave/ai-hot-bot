# 如果你最近晋升答辩，自查以下准备做没做到

**分类**：problem-solution-list
**主题**：晋升答辩
**风格**：干货类、方法论
**来源**：小红书 @夏树
**添加时间**：2026-01-30

---

最近又到晋升季，每年很多同学熬了几年，好不容易被提报，材料改了N遍还是没底，最后没通过我五年每1-2年晋升一次，把答辩必过SOP分享出来，家人们先收藏再看：

先说说晋升失败的3个原因，家人们自查一遍：
1、写成绩效总结：还是罗列工作，"做了什么活动、上线什么方案"没体现自己的思考
2、工作价值没讲透：要么满是黑话评委听不懂，要么没核心数据，看不出贡献
3、能力不匹配：比如升管理岗还在讲执行细节

答辩通过的唯一秘诀：向评委证明，你已经做下一职级的工作很久了

一、开头2分钟：让评委给咱贴标签（个人履历）
🎮就像玩王者的英雄，你是辅助还是射手，让评委快速给你打标签，2句话+1张业务逻辑图就够：

1、过往经历：部门+过去公司+绩效荣誉+相关特长

2、负责模块及价值：你负责什么，并用通俗的话讲明白作用，比如"作为电商推荐策略产品，负责将商品供给和用户需求进行供需匹配，最终带动GMV增长"，评委马上get

二、核心10分钟：点亮能力技能树（工作产出）
家人们别把过去几年的项目全堆上去，妥妥的流水账。还是🎮像给英雄点技能点一样：
1、先拿能力模型：找HR或+1要下一职级标准
比如P7👉P8: 带方向、做规划，IC（个人贡献者）变leader

2、用把项目按主线串起来，不超过3条，举个🌰：
- 主线1：把现有类目做深做厚（规划和方向）
- 主线2：0→1搭建商家成长体系（创新突破）
- 主线3：跨部门统筹大促专项（协同影响力）

3、每条主线，虚实结合：
- 务虚：讲背景→ 机会→ 思考→ 结果（核心数据+行业/公司内对比），比如"之前中小商家动销率仅x%，发现商家缺乏xx，于是设计xx，最终动销率提升xx%，远超行业xx%的水平"
- 务实：挑1个重点项目讲细节，运营说玩法设计，产品说功能流程，数据图、对比截图，感染力[棒R]

三、结尾3分钟：给评委画饼（未来规划）
很多同学会忽略，评委们听我们讲了十几分钟，信息太多，得先做个总结，唤醒评委记忆。再讲规矩，别喊空口号，和上面的工作产出类似，解决什么问题、具体可行手段、可能风险和应对

按这个SOP准备，晋升成功就是水到渠成的事[亲一个R]
