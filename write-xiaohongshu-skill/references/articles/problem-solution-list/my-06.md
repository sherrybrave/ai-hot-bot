# 大厂评委全票通过的顶尖晋升答辩

**分类**：problem-solution-list
**主题**：晋升答辩、方法论
**风格**：干货类、SOP
**来源**：小红书 @夏树
**添加时间**：2026-01-30

---

我工作5年的时候，靠自己的努力终于在大厂晋升管理岗，不仅评委全票通过，还对我的业务表达了合作兴趣，后面有了进一步的合作。当时在网上搜到的信息都比较碎片，把晋升的顶级答辩思路和每一part的材料撰写要点分享出来，想晋升的宝子们一定马住：

一、个人履历：简单开头即可，不超过2分钟
内容：做过什么业务，过去绩效如何，获得什么荣誉，有什么擅长
格式：一张业务逻辑大图+文字，介绍自己负责什么业务，在整个部门和公司的位置、价值，让评委带入

二、工作产出：重中之重！给我用心写！10分钟左右
表述逻辑：
- 是把工作的【复杂度、体系化、开拓性】讲出来，适度故事包装，有起承转合、具体事件，而非像OKR总结一样平铺直叙
- 可以思考我来之前现状如何，来之后如何明确业务方向，有哪些成果，体现个人独特价值
具体这么来写：
1、有重点，把过去的工作总结为2-3条主线，每条主线差异化，举例：
- 主线一：该工作的体系化、全面性、复杂度
- 主线二：0->1落地xx功能，创新设计方案->协同多团队落地->取得好结果
- 主线三：0->1落地xx方向 ，牵头落地创新功能->建立指标体系->取得超预期结果
2、具体表述每条主线的时候，虚实结合：
- 务虚：概述每条主线的来龙去脉
  - 当时的业务背景是什么，业务发展遇到了什么问题，自己发现了什么机会
  - 做了什么动作，如何分析思考，以业务逻辑图的方式体系化地写出来
  - 最终取得什么结果，业务指标的AB实验收益、AA增长
- 务实：每条主线挑1-2个亮点项目展开讲细节
  - 核心功能流程、交互如何设计
  - 结果以有感染力，用数据增长图、迭代前后录屏/截图、showcase等方式展现
  - 如果是晋升3-1，还要写如何虚线管理带队

三、规划与展望：收尾、升华，不超过3分钟
总结过去：过去几年，带领业务方向完成几次演进不断前进，提炼成几个阶段+下一阶段
展望未来：对于未来自己负责方向的展现与规划，分成几条主线，可行动作、可能风险、如何应对，突出转变突破
