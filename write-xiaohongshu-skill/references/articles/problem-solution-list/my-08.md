# 被打低绩效，除了伤心愤怒，我们还能做什么

**分类**：problem-solution-list
**主题**：低绩效、应对策略
**风格**：干货类、情绪共鸣
**来源**：小红书 @夏树
**添加时间**：2026-01-30

---

最近各大厂年中绩效开奖，行情不好，低绩效比例变多。作为一个心理学出生的大厂人，我非常感同身受，在领导宣告你的绩效m-、325、C之后，涌上来的伤心、委屈、愤怒的情绪。但还是希望，大家不要太过于沉浸到低落情绪里，坚强起来争取自己的利益。

身边有很多同事朋友都拿过低绩效，我总结了他们的经验，这样应对可以利益最大化：

❤️首先是心态。拿低绩效有预兆，领导可能提前几个月表现出不满或冷处理。要明白，绩效是管理手段，团队里低绩效有比例强制分配，而且是领导主观决定，说明你和ta不合适，不代表你能力差

然后分析处境，三种情况：
1、确实产出低：团队里他人的活更好、做事更投入，产出比你多且好。这种情况有得救😄
2、和领导相处超过1年：在团队扎根深，但因为产假、领导端水等原因轮到背锅。也有得救🥸
3、自己刚来或者领导刚来不到1年：相处时间短，合作关系没建立起来，变成了弃子，果断走人😡

最后是应对策略：
前两种情况：虽然有得救，但也得两手准备：
1、👉认真改进，逆风翻盘：重建与领导的信任。和+1、+2分别1on1沟通，难得的机会，聊清楚领导预期，哪里做的不好，标准什么样，有什么建议，帮自己制定一个绩效改进计划，落地拿结果。平时和+1、+2搞好关系，做好一个演员的自我修养🧐，展现上进心与忠诚度

2、👉苟着，找活水或者下家：如果内部转岗没有限制绩效，尽量转岗。和之前合作过的领导打听机会，有信任基础更容易通过。同时准备好简历，多加几个猎头看外部机会

最后一种情况：要赔偿、找下家：
👉网上找靠谱律师咨询，了解如何争取赔偿。收集证据，比如之前领导布置没有明确预期结果、事后追责，或前期未定目标后面追加目标等。准备充分后，找领导和HR申诉，要求赔偿💰。对方同意后在赔偿方案争取1-2个月缓冲期，用来找下家

低绩效不是判决书，与其困在情绪里内耗，不如用谋略来为自己争取逆风翻盘的底气。我那些拿低绩效的朋友，最后过得也都不错
