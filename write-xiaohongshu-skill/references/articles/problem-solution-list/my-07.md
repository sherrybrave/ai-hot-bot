# 又到绩效期，90%人不知道的好绩效潜规则

**分类**：problem-solution-list
**主题**：绩效、职场潜规则
**风格**：干货类、观点类
**来源**：小红书 @夏树
**添加时间**：2026-01-30

---

又到了各大厂绩效期，大家都知道，每年有2次绩效期，年终一次、年中一次。在大厂优秀员工很多、标准也很严格，写绩效如果能突出这四点，拿高绩效可能性更大。

1、每个人都会写收益，但故事线的差异价值是核心

几年下来，我给超过200个同事写过评价。感触最深的是，每个人的工作产出里都有非常多的数据收益，看多了都会觉得有些视觉疲劳。一方面，大厂对工作结果的数据评估意识非常强，另一方面，也说明大厂优秀的人实在太多了。

我观察了不同绩效同事写的工作产出和自评，总结"有没有体现你的故事线有什么差异价值"正是绩效是否好的分水岭。

当然，这里还有一个关键因素是嫡系，这是另一个话题了。

2、不要流水账，要写得重点突出、主次分明

很多同事在工作产出里，都是在记流水账，把过去半年、1年的事情列了出来。如果你不是很了解ta的工作，很难看出来ta的工作产出是什么水平。

写得最好的同事，会把每块工作的业务背景、业务逻辑、难点攻破、业务价值都讲得非常清楚明了，让你一眼就能看出来，这是个产出硕果累累的优秀同事。

3、邀请同级同事评估人有讲究

同级同事可不能随便邀请，平时合作不愉快的不要邀请，合作少的也别邀请。老板也不是傻子，平时周报、汇报大概知道你和哪些人合作比较多，看到合作少的被邀请人，也知道大概是你找一些搭子互相吹捧。

建议多邀请不同团队、不同职能角色的同事，甚至合作团队的+1、+2的leader，能全方位体现你优秀的跨团队沟通协同能力。

4、公司价值观，其实没那么重要，但也别成为扣分依据

公司价值观确实没那么重要，你的绩效是工作产出和是否嫡系决定的。但是如果有什么问题，比如被合作方投诉、工作情绪化，或者有什么不尽职的案例，老板也有充分的理由可以给你的绩效扣分了。
