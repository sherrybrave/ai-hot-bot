# 大厂绩效：停止无效努力，玩懂存量博弈

**分类**：problem-solution-list
**主题**：绩效、底层逻辑
**风格**：观点类、深度分析
**来源**：小红书 @夏树
**添加时间**：2026-01-30

---

很多新人尤其是一路上来的好学生，误以为绩效是公平考试，努力就能拿奖学金。扎心的事实是有人埋头苦干却拿低绩效，有人看似轻松却年年优秀。

大厂绩效不是对勤奋者的奖励机制，而是一场多方博弈的存量游戏：

1、真相：你拿到的绩效，不是评出来的
绩效结果早在你写自评前，就在博弈中成型了：
- 自评与环评：你翻周报、列数据，绞尽脑汁，但leader早就心里有数了。360环评若出现一个负面评价，可能会被作为给你的低绩效的证据
- leader定调：这是最关键的一环，+1判断你贡献的价值和团队平衡，给出初版
- 绩效纠偏：+2与HRBP会从团队整体层面审视，哪怕+1想挺你，但如果+2想提拔谁，或者强制绩效分布，你的b+变b，b变b-也常见

2、尺度：价值远比产出更值钱
无论是OKR还是KPI，本质都一样。完成率只是门槛，真正的差异在于贡献度和突破。做十个小优化，价值远不如解决一个困扰团队已久的难题，或是在关键指标上创造了增长

3、Leader视角：绩效是管理工具，而非奖励机制
很残酷，大厂绩效通常有强制比例，一般优秀（e/m+）占比不超30%，较差（m-及以下）约20% 。这意味着不是你做得好就能拿高绩效，而是要看相对排名

Leader在分绩效的时候，考量的是管理目的：
- 平衡核心利益：如果A同学刚拿过高绩效，而B的业务刚突破，leader可能会把名额给B，留住人才
- 绑定核心OKR：谁的工作支撑了leader今年的核心KPI，绩效自然优先倾斜
- 末位淘汰与激励：给有潜力的员工高绩效以示鼓励，给老人打低绩效逼其自行离职，实现汰换
之前和我的+1聊过，他说得很直接："绩效就是管理团队的工具，把这潭水搅动起来。"

4、公司视角：用最小成本驱动最大价值
公司用不超过30%的低比例的高绩效和有限的奖金（3个月以上月薪），就驱动了剩余70%的人在降本增效的压力下疯狂竞速。这套逻辑利用了你的晋升焦虑，让你在不增加公司太多成本的前提下，自愿贡献最大化价值

我想对正在焦虑绩效的你说几句真心话：
1、停止无效努力，只做能写进leader OKR的工作
2、维护口碑，环评决定你的下限
3、提前半年和leader对齐晋升与绩效标准

绩效不是我们人生的标准，用更少的精力玩懂存量博弈，更多的时间放到真正的自我提升和生活上。
