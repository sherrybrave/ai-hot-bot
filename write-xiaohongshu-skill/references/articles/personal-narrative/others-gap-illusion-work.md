# Gap期间的最大错觉：我可以上班了

**来源：** gavin嘎嘎 | 小红书
**链接：** https://www.xiaohongshu.com/discovery/item/698477ef000000000e00e077
**提取日期：** 2026-02-18

---

裸辞后，一股脑去了广西、西藏、泰国、法国、西班牙。期间经常有hr来加我（主要是宇宙厂），我都直接回绝说，"不找工作"。

但两个月过去后，我开始松口了，"可以面面看" -- 我突然觉得自己可以回去工作了。

我的脑子跟自己说：
"这么玩下去好像也有点无聊"
"大厂实在是赚太多了，再赚几年吧"
"这个岗位看起来不会那么坑"
"万一遇到一个不卷的好老板呢？"
"周边的人都在工作诶！"
"好像也休息得差不多了，恢复能量了！"
.....

于是，我就开始接受了一个面试；于是，我就发现......我大错特错了！

当你开始重新接触"工作"，哪怕只是面试聊天、只是一次和前同事的讨论、只是整理之前的项目经历，你就像是普鲁斯特笔下的贡布，以椴花茶泡玛德琳蛋糕为触发点，唤醒了脑子对往事的全部记录和感知 -- 只不过，带来的完全不是快感

1️⃣开始面试后，有一次和还在前司的朋友聊天，问了一嘴公司的情况。结果对方开始源源不断地输出，给我讲了换新老板的故事、绩效沟通的故事、内部分组"赛跑"的故事、职场小人的故事......配着阿卡和德约决赛的画面听这些，我整个人已经生理性紧张到不行了

突然意识到，我对互联网公司非人环境的容忍度并没有提升，我对优绩主义虚无缥缈的职场也没有更加能够适应，我只是忘了，我只是选择性美化了那些回忆。

2️⃣面了一个很适合的岗位，面完负责人后hr说："面评挺好的，但你是第一个进终面的，还得面2个比较一下"。我开始紧张内耗，晚上失眠到了三点，一直复盘自己表现的如何、对方给出的"微反馈"如何

这种焦虑出现在我过去每次评审、汇报、述职、晋升时，我还是那个容易紧张的人，容易给自己制造很高的预期，又同时很在意别人看法 ，且一直希望拿到更好的结果 -- 这并没有因为我休息了两个月，而发生根本的改变。

我再次停止了面试。我还是没有找到可以回互联网上班的状态，也没有想清楚是否要回到那套评价体系里，更没有信心可以再过几十年大厂打工的生活。

但是我也准备停止无止境旅行，去回到生活里，试试不一样的事情。基于自己的兴趣爱好，学习新的东西、试试不一样的行业。

期待新的一年

## 框架分析

**框架类型：** 个人叙事 + 转折反思

**结构：**
1. **背景铺垫** - 裸辞旅行，拒绝工作邀请
2. **心态转变** - 松口面试，自我说服的理由
3. **情节转折** - 开始面试后发现问题
4. **具体案例** - 两个场景（前同事聊天、面试内耗）
5. **自我洞察** - 认识到自己没有改变
6. **最终决定** - 停止面试，尝试新方向

**写作特点：**
- 真实经历为基础
- 心理描写细腻
- 对话式引用（引用自己的内心对话）
- 具体场景支撑观点
- 文学化比喻（普鲁斯特贡布）
- 表情符号使用
