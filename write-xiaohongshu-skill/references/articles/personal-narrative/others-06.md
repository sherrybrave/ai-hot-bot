# 我的12年职场路，4个重要认知

**主题**：职场经历、成长认知

**来源**：别人

父亲的坚持让我看到了更大的世界:
父亲是普通工人，6岁时开始培养我画画，坚信我能考上中央美院。尽管遭亲戚和老师的质疑，但他的信念始终未变。终因坚持，我以第30名的成绩实现了父亲的愿望。

人生充满波折，每一次跌宕，都是洗尘重启

➀设计师需要独立思考 - 专业能力会在绝望后实现真正的跃迁

毕业一年内我换了四家公司，仅因领导审美差。第二年加入一家年轻的互联网创业团队，与一位产品VP一起工作。他对于设计的要求很高，每次过稿的时候他只会默默的摇头，且不会明确的告诉我如何改进。我每天工作到深夜，在无数次否定后，我选择继续努力。有一次，他看过我的方案后说："这个不错，就这样吧！"那一刻打通了很多设计和产品在思维层面的壁垒。这段经历给予我专业思维上的转变，让我学会了主动思考，寻找答案，并创新性地解决问题。

➁工作6年的我发现管理能力为零--管理是一个需要系统学习的专业

当时的产品火遍全网，获得了一定的专业影响力。应小米邀请，负责管理电视业务设计团队。每天像保姆一样指导设计师工作，保证好的设计落地，却无法有效向上汇报，管理领导的预期。当管理职位被取代时，我感到委屈和震惊。经过反思和调整，研究管理，才明白管理有其自身的知识和方法。

➂无法忍受不涨薪的工作 - 人要走出舒适区迎接新的挑战

在小米互四部，我成功组建新团队并培养优秀设计师，得到高层认可。就这样轻松愉快的过了两年，当时32岁年包180w。但整体的业务团队缺乏互联网思维。我清楚记得，当时阿里的设计师不仅能做设计还有很多方法论与业务的思考，这让我感到陌生和危机，加上小米薪资涨幅明显低于市场水平。于是下定决心加入快手。我知道很可能会失败，但我要迎接新的挑战！通过一年的研究和学习，我掌握了互联网思维，这对未来职场和创业中的产品运营工作非常重要。

➃短暂的打工生涯，是为了长久的经济自主

在快手的工作得心应手，管理80人团队，取得了很多成绩如搭建高价值礼物体系，组建设计海外团队与海外产品。但空降的领导带来的职场纷扰让我决心告别。在小红书从创新项目夭折到UED中心直播体验负责人。为自己未来的5年打好了基础，决定在当下离开职场，转向多个领域市场去发展。现在，我是一家AI科技公司的合伙人，负责设计和运营多个产品。也希望给更多的设计师和企业创造价值，愿能毫无保留将我的经验分享给你们。

2023 砥砺前行，开启新征程。
