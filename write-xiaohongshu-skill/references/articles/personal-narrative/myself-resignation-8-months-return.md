# 写在除夕：31岁大厂裸辞8个月，要回去上班了

**来源：** 夏树Sherry（作者本人）| 小红书
**链接：** https://www.xiaohongshu.com/discovery/item/6991b0b80000000015031701
**提取日期：** 2026-02-19
**备注：** 这是大小姐自己的爆款笔记，非参考文章

---

大家好，我是树，这是我裸辞的第8个月。

我之前在互联网大厂工作了6年，第五年晋升到3-1，年薪百万。虽然做的产品让我很有成就感，但复杂的职场环境和压力让我非常内耗，严重的时候连续两个月晚上失眠到凌晨四五点。

身边有不少朋友说我适合创业，单干，于是我在去年6月底从上家公司离职了。

离职前，我对裸辞后生活有美好的向往，趁着在职办了很多签证，计划在云南、东南亚、欧洲、非洲、美国各待一个月，一年内有一个万粉的账号，并且月入至少1万。

可后面这8个月的生活，和我想象中完全不一样。

在云南旅游第一个月的时候，我就开始焦虑了。在静寂的哈巴雪山下，我居然脑子里还惦记着创业。于是，云南之行结束后，我就回家开始做自媒体。

裸辞的生活比上班还忙：每天早上8点钟起床，睁眼就是干，晚上忙到10点，周末也经常没休息。我从头开始学：怎么找对标怎么选题怎么拆爆款怎么写脚本怎么剪辑。虽然收入基本能cover生活费，但是我始终觉得职场知识付费不是我的长期方向，离开职场后，很难持续更新下去。

这个时候我很迷茫，仍不清楚创业方向是什么。我想过出海、跨境电商、户外游，但始终停留在想。

最迷茫的时候，朋友邀请我去非洲，于是我又玩了一个月，回来后开始做电商。

选品，跑线下批发市场，在1688翻遍了供应商，拿货，做内容，上架商品。在这个过程中，我才发现原来找到一个适合自己的商业模式并且艰难的啃下来有多不容易。我是一个不太注重物质生活的人，研究商品细节和卖点，让我非常痛苦，也打动不了用户。很快，半个月后，我把店关了。

但在这个过程中，我把号做起来了，跑了一遍电商链路，还用AI做了一些工具。机缘巧合，猎头打来电话，推了一个电商产品岗位。面了差不多五六轮，我顺利拿到了offer。

裸辞的第8个月，我终于明白：裸辞并不重启人生。当你在打工的时候没有自己的商业模式，裸辞之后也很难从0跑通。

重新打工后，我希望接下来的3年能够：
1. 建立一套自己的人生评价体系，不困在外在评价体系里
2. 靠近一个生意，把一个商业模式摸清楚并能跑通
3. 拥抱AI，用好AI，多多做side project创造价值

这是我的裸辞故事，也是重新出发的起点，你呢？

## 框架分析

**框架类型：** 个人叙事 + 反思成长 + 目标展望

**结构：**
1. **开篇钩子** - 直接点明核心信息：裸辞8个月
2. **背景铺垫** - 大厂6年、年薪百万、职场压力大
3. **美好设想** - 离职前的计划（环游世界+自媒体创业）
4. **现实打脸** - 8个月的坎坷经历（云南焦虑→自媒体→电商）
5. **核心感悟** - 金句总结：裸辞并不重启人生
6. **未来规划** - 3个具体目标
7. **结尾互动** - 开放式提问，引发共鸣

**风格特点：**
- 真诚不造作，有具体时间、地点、细节
- 有反转：从"裸辞自由"到"选择打工"
- 有深度思考：不是简单分享，而是有反思和成长
- 情感共鸣：引发读者的思考和讨论

**爆点要素：**
- 大厂背景（3-1、年薪百万）→ 身份认同
- 裸辞8个月 → 时间跨度，引发好奇
- 真实经历（云南、自媒体、电商）→ 具体可信
- 反转结局（重新打工）→ 意外但有理有据
- 深度金句（"裸辞并不重启人生"）→ 引发思考

**写作技巧：**
1. **时间线清晰：** 从离职前→8个月经历→现在→未来
2. **对比强烈：** 美好设想 vs 现实打脸
3. **细节丰富：** 具体地点（哈巴雪山、非洲）、具体行为（跑批发市场、1688）
4. **情感真实：** 不美化经历，坦诚痛苦和迷茫
5. **结尾升华：** 从个人经历上升到人生思考
6. **开放式结尾：** "你呢？" 引发读者参与讨论

**字数控制：** 约800字，信息密度高，节奏紧凑

**适用场景：**
- 年度总结/除夕夜文案
- Gap经历分享
- 职场/创业反思
- 个人成长记录
